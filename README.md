Brainf*ck のJITコンパイラ
====

自分がJITコンパイルを学習するモチベーションになればと思い、
Brainf*ck の単純なインタプリタと、それを少しだけ最適化したバージョン、そしてJITコンパイルで生成した機械語を実行するバージョンを用意し、
それぞれの実行時間を比較してみました。

動作確認は WSL(Linux x86_64) のみです。
Windows だとABIが違うのでJITバージョンが動かないです。

run.sh を実行すると、3つのバージョンの実行時間を出力します。

私の環境では初期バージョンが約25秒に対し、JITコンパイルするものだと1秒弱になったので、25倍の高速化を実現できました。
